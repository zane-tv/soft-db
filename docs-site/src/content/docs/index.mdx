## Why SoftDB?

<CardGrid>
  <Card title="Cross-Platform" icon="laptop">
    Native desktop app for Windows, macOS, and Linux. Built with Go + React via Wails v3 — fast and lightweight, no Electron overhead.
  </Card>
  <Card title="7 Databases, One App" icon="list-format">
    PostgreSQL, MySQL, MariaDB, SQLite, MongoDB, Redshift, and Redis. No more switching between different tools.
  </Card>
  <Card title="AI-Powered" icon="star">
    Built-in AI assistant understands your schema. Powered by OpenAI. Get context-aware SQL instantly.
  </Card>
  <Card title="MCP Server" icon="rocket">
    Built-in MCP server lets Claude, Cursor, and Windsurf query your databases directly. One config line to connect.
  </Card>
  <Card title="Secure by Default" icon="approve-check">
    AES-256-GCM encrypted credentials, parameterized queries, OAuth PKCE, and SSH tunneling support.
  </Card>
</CardGrid>

## Core Features

### SQL Editor

Full-featured Monaco editor with syntax highlighting, schema-aware autocomplete, multi-tab queries, and integrated snippets.

### AI Assistant

Chat with an AI that knows your database schema. Use @table mentions to include column metadata. Powered by OpenAI with multiple model options.

![AI Assistant](../../assets/screenshots/ai-chat.png)

<LinkCard title="AI Assistant Guide" href="/features/ai-assistant/" description="Setup providers, use schema context, optimize queries" />

### Data Grid & Table Explorer

Virtualized data grid with inline cell editing, sorting, filtering, and pagination. Edit data directly — changes batched as parameterized UPDATE statements.

### Visual Tools

<CardGrid>
  <Card title="ER Diagram" icon="open-book">
    Auto-generated entity-relationship diagrams from FK metadata with ELK auto-layout.
  </Card>
  <Card title="Query Builder" icon="pencil">
    Build SELECT queries visually — pick tables, columns, conditions, ordering without writing SQL.
  </Card>
  <Card title="Query History & Snippets" icon="document">

  </Card>
  <Card title="Import & Export" icon="right-arrow">

  </Card>
